90sStyle LimeChat Theme

## Install

 cd ~/Library/Application Support/LimeChat/Themes
 git clone git://github.com/mephux/90sStyle.git

## What is Limechat?

Limechat is a small IRC-client for Mac OS X, you may find it at http://limechat.net/mac/.
